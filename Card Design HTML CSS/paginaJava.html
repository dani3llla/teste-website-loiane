Começar

Sobre o Curso

O curso Java Básico e Orientação a Objetos é ideal para iniciantes ou quem quer reforçar a base em Java. Você aprenderá desde os conceitos iniciais até os fundamentos da Programação Orientada a Objetos, com conteúdo claro, prático e acessível mesmo para quem nunca programou.

O que você vai aprender?

  • Instalar e configurar o ambiente de desenvolvimento Java.
  • Entender a estrutura básica de um programa em Java.
  • Trabalhar com variáveis, tipos de dados e operadores.
  • Criar estruturas de decisão e repetição:
    • if
    • switch
    • loops (for, while, do-while)
  • Manipular vetores e matrizes.
  • Realizar depuração de código com o For each e ferramentas de debug.
  • Aplicar os conceitos de Programação Orientada a Objetos (POO), incluindo:
    • Classes e Objetos
    • Encapsulamento
    • Herança
    • Interfaces
    • Polimorfismo
  • Tratar erros e exceções de forma eficiente.
  • Baixar o código-fonte de todos os exemplos vistos em aula.
  • Obter um certificado de conclusão gratuito ao final do curso.
